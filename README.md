This is the source code for [Screenshot.rocks](https://screenshot.rocks?utm_source=github)

## Installation :cd:
1. `git clone git@github.com:daveearley/screenshot.rocks.git`
2. `cd screenshot.rocks`
3. `yarn install`
4. `yarn start`
